# TODO Dashboard

This is a TODO dashboard for TDCX frontend assessment.

## Installation

1. Clone the project

```
git clone https://github.com/mirza-sync/todo-dashboard.git
```

2. Install dependencies and run the project

```
npm install
npm run dev
```

3. Go to http://localhost:5173/
